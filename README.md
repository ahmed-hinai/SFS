# SFS - Show Fan Speeds

SFS is a TUI for nbfc-linux written in C. 


## Features

- **Fan Speed Monitor**
- **Temperature and Utilization Monitor**
- **Utilization Stats**

## Installation

1. Install [nbfc-linux](https://github.com/nbfc-linux/nbfc-linux).

2. Clone and install

  ```bash
git clone https://github.com/ahmed-hinai/SFS.git
   ```

  ```bash
cd SFS
  ```

  ```bash
make install
  ```
